Diagnóstico da Aplicação React

Esta página ajudará a diagnosticar problemas com a aplicação React do Keeper of Realms Forge.

Problema: Página em Branco

Se você está vendo uma página em branco ao acessar a aplicação, isso pode ocorrer por vários motivos:

  1. Acesso incorreto: Você pode estar acessando um arquivo HTML estático em vez da aplicação React.
  2. Problemas de roteamento: O React Router pode não estar configurado corretamente.
  3. Erros JavaScript: Pode haver erros no código que impedem a renderização.
  4. Servidor não iniciado: O servidor de desenvolvimento Vite pode não estar rodando.

Atenção: Acessar diretamente arquivos como /home.html, /shop.html, etc. não carregará a aplicação React completa, resultando em telas em branco ou comportamento inesperado.

Solução Recomendada

Para acessar a aplicação React completa: Use sempre a URL base http://localhost:3000/ e navegue usando os links internos da aplicação.

O projeto Keeper of Realms Forge tem duas abordagens diferentes:

  1. Páginas HTML estáticas na pasta public/ - estas são páginas simples que não fazem parte da aplicação React principal.
  2. Aplicação React com sistema de rotas - esta é a aplicação principal que deve ser acessada pela URL base.

Passos para Resolver

  1. Certifique-se de que o servidor Vite está rodando com npm run dev
  2. Acesse http://localhost:3000/ (não adicione /home.html ou outros arquivos)
  3. Se o problema persistir, verifique os erros no console do navegador (F12)
  4. Tente limpar o cache do navegador ou usar uma janela anônima
